# TGallery

Minimalist fotoğraf ve video galerisi. Kotlin + Jetpack Compose.

## Özellikler

- Tümü / Albümler görünümü
- Pinch-to-zoom fotoğraf detay
- Video oynatma (ExoPlayer)
- Coil ile hızlı thumbnail yükleme
- Mplayer ile aynı siyah/beyaz tema

## Build

```
./gradlew assembleDebug
```
